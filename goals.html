Consistency Growth Discipline

Rule

Small wins every day > random big bursts.

Focus

Do less, better. Then scale up.

Short-term (next 30–90 days)

  • Finish and polish this website + keep it updated
  • Build 2–3 projects I can showcase on the Work page
  • Improve cybersecurity fundamentals (networking + security basics)
  • Create a weekly routine I can stick to

Long-term (6–24 months)

  • Strong portfolio and a clear path in tech/cybersecurity
  • Earn certifications and prove skills with projects
  • Build confidence speaking about my work
  • Keep improving: health, discipline, and mindset

Habits I’m building

Simple things that compound.

Daily

1 focused session of learning or building (even if it’s short).

Weekly

Ship something small: update site, finish a mini project, or write notes.

Monthly

Review progress, remove distractions, and set the next targets.
